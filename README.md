# CORD-19 Data Explorer

This repository contains a **Streamlit web application** to explore COVID-19 research papers using the CORD-19 dataset.

## Project Overview

The project demonstrates:

- Loading and exploring a real-world dataset (`metadata.csv`)
- Data cleaning and preparation
- Basic analysis and visualization
- Interactive Streamlit application for exploring publications by year, journal, and source
- Word cloud visualization of paper titles

## Dataset

The `metadata.csv` file comes from the [CORD-19 dataset](https://www.kaggle.com/allen-institute-for-ai/CORD-19-research-challenge).
